# 工具栏与自动发送

工具栏用于在企业微信聊天侧辅助销售、客服或运营完成跟进动作。自动发送能力用于减少重复点击，但需要先完成客户端 ID 绑定。

## 工具栏能做什么

- 查看客户基础信息和最近互动摘要。
- 记录跟进结果，沉淀到客户中心。
- 查看任务或建议动作。
- 在满足条件时开启自动发送。

## 自动发送前置条件

### 先绑定客户端 ID

1. 当前账号已经绑定 MuYuAI 客户端 ID。
2. 企业微信客户端已打开对应聊天窗口。

### 再确认任务状态

3. 自动发送面板状态正常。
4. 当前任务允许自动发送。

## 绑定客户端 ID

进入自动发送面板后，如果还没有绑定客户端 ID，系统会提示你填写。没有绑定时不能开启自动发送。

> 提示：客户端 ID 是用于识别当前 MuYuAI 客户端的安全标识，不需要理解底层技术字段。

## 使用边界

- 不要多人共用同一个客户端 ID。
- 如果更换电脑或客户端，需要重新确认绑定信息。
- 自动发送过程中如果状态异常，应先停止自动发送，再检查客户端状态。
- 自动发送适合高重复场景，复杂对话仍建议人工确认。
