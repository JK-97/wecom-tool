# 工具栏与自动发送

工具栏是在企业微信聊天侧使用的轻量工作区。它不是后台客服中心的缩小版，而是帮助销售、客服和运营快速识别客户、查看建议、记录跟进，并在合适场景启用自动发送。

## 适合谁看

- 在企业微信聊天侧处理客户的销售、客服和运营。
- 需要使用自动发送减少重复操作的一线成员。
- 需要排查客户端 ID、自动发送状态或多会话选择问题的管理员。

## 操作前准备

1. 确认你在企业微信客户端中打开了客户聊天窗口。
2. 确认当前账号已经绑定或准备绑定 MuYuAI 客户端 ID。
3. 确认当前任务适合自动发送，复杂沟通仍应人工确认。

## 工具栏能做什么

- 查看当前客户和最近互动摘要。
- 在多微信客服账号场景下选择本次要辅助的会话。
- 查看 AI 建议、任务提示和跟进信息。
- 绑定 MuYuAI 客户端 ID，并在允许的任务中开启自动发送。

## MuYuAI 客户端 ID 是什么

MuYuAI 客户端 ID 用于识别当前电脑上的 MuYuAI 客户端。没有绑定时，系统无法确认哪个客户端负责执行自动发送，因此不能开启自动发送。

### 绑定规则

- 每个使用自动发送的成员都需要绑定自己的客户端 ID。
- 不建议多人共用同一个客户端 ID。
- 更换电脑、重装客户端或切换工作环境后，需要重新确认绑定。
- 页面展示时应使用“MuYuAI 客户端 ID”的业务名称，不直接暴露底层字段名。

## 使用流程

1. 在企业微信中打开客户聊天侧边栏。
2. 进入工具栏，确认当前客户和会话。
3. 如果提示绑定 MuYuAI 客户端 ID，先填写并保存。
4. 查看摘要、建议或任务要求。
5. 确认内容适合自动发送后再开启。
6. 自动发送过程中观察状态；异常时先停止，再检查客户端状态。

> 注意：自动发送适合高重复、低歧义的任务。复杂谈判、投诉、退款、价格承诺等场景仍建议人工确认后发送。

## 完成后检查

1. 自动发送状态已经显示为完成、停止或可重新开启。
2. 当前聊天窗口中的发送结果符合任务要求。
3. 如果中途暂停或失败，先确认客户端状态，再决定是否重新开启。

## 多会话选择

同一个客户可能近期命中过多个微信客服账号。工具栏无法总是自动判断你当前想辅助哪一个会话，因此会在有歧义时要求先选择会话。

### 选择时重点看

- 微信客服名称。
- 会话状态。
- 最近消息。
- 最近活跃时间。

## 常见问题

### 为什么一进自动发送面板就弹出客户端 ID

这是正常保护。没有客户端 ID 时，系统不能确认自动发送执行者，因此必须先填写，且不能直接开启自动发送。

### 已绑定后还要求确认

可能是当前账号、更换电脑或客户端状态发生变化。先确认展示的客户端 ID 是否属于当前使用环境。

### 自动发送中断

先停止当前流程，再检查企业微信聊天窗口、MuYuAI 客户端状态和任务是否仍允许自动发送。
